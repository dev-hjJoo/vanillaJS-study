## 본 디렉토리의 역할
* 아주 기초적인 문법을 다루는 강의이므로 꼭 필요한 부분만 기록
* 코드가 필요한 부분은 세부 디렉토리를 형성하여 코드 작성
* 코드가 필요하지 않은 부분은 notion에 부분별 정리
* 번호는 <b>'자바스크립트 배우기 전 꼭 봐야 할 영상 …'</b> 강의를 0으로 지정한 후 순차적으로 증가시켰음!
